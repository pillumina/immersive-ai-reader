# Design Context

## Users

学术研究者和普通读者兼有。核心场景是 **PDF 深度阅读 + AI 辅助理解**：选中文本高亮、做笔记、与 AI 对话探讨、Focus Mode 专注阅读。用户在长时间阅读时需要沉浸感，在需要 AI 时需要流畅的上下文衔接。

## Brand Personality

**精致沉浸、温暖编辑感**。界面应让人觉得：这是一款为认真阅读而生的工具，值得长时间使用。视觉上克制但有温度（Warm Editorial 暖色调），交互上流畅但不炫技，动效有目的性——帮助用户理解状态变化，而不是增加噪音。

三个关键词：**沉浸**（无干扰阅读）、**精致**（细节打磨）、**温暖**（非冰冷工具感）。

## Aesthetic Direction

**Warm Editorial** — 暖琥珀色主色调 + 石英灰背景，Plus Jakarta Sans + Newsreader 字体组合。参考 Notion 的克制感和 Linear 的精致感，但更温暖（暖色调色板而非冷色）。已有三个主题（Warm Light / Dark / Warm Dark）完整实现。

**Anti-references:**
- 不要渐变文字、霓虹色、赛博朋克风
- 不要过度使用 glassmorphism
- 不要 bounce/elastic 动效曲线（感觉过时）
- 不要 AI 配色（深色+青色/紫色渐变）

## Design Principles

1. **动效有目的** — 每个动画回答一个问题：状态变了什么？为什么变？无目的的装饰性动画宁可不要。
2. **聚焦沉浸感** — 阅读时隐藏 UI 噪音；Focus Mode 是核心体验，进入时应有仪式感。
3. **温暖克制** — 暖色调贯穿所有状态；主色琥珀色用于关键操作和焦点，不滥用。
4. **60fps 优先** — 所有动画基于 `transform` + `opacity`，不使用布局属性动画。reduced-motion 用户应能完整使用。
5. **内容优先** — PDF 内容永远第一；UI 元素（面板、工具栏、空状态）服务内容，不喧宾夺主。
